AnalysisDebug

This website contains the output of running php5analysis with file tests/functions/fac.php.

Analyis contains the source code pretty printer internaly (with some bugs) and also the output of the monotone framework in an annotated control flow graph.

Debug contains additional information such as the parser (sglri) result, the abstract syntax tree graph and internal information concerning the control flow.

For more information on the internals consult the github page.
